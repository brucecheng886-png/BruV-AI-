## 【現有安裝的快速補救】（跳過重裝，手動補下載）

> 如果已經用舊版裝過，先跑這條指令把模型補進 Docker Ollama，不需重裝：

```powershell
docker exec bruv_ai_ollama ollama pull qwen2.5:14b
```

等下載完成（約 9GB）後直接跳到 **E 部分**驗證。
