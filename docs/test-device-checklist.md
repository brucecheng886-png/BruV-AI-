---

## 除錯流程

```
收到錯誤回報
    ↓
在 Dev 端重現這個錯誤
    ├─ 找到 → 直接修改
    └─ 找不到 / Dev 測試通過 → 向使用者提出需要的詳細資訊
```

## 回報格式

回報時請說明：
- 哪個裝置（我的電腦 / 我的筆電 / 測試員）
- 發生了什麼問題（錯誤訊息或描述現象）

Copilot 會先檢查開發端，确定是否為開發端問題，再决定是否需要向測試端索取詳細資訊。
